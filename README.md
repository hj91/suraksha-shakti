suraksha-shakti
===============

"suraksha" means safety and "shakti" means power in india. This app is used for sharing android gps location on the move.

While this app works perfectly in good conditions where GPS visibility and network is stable, there are some practical issues that are bit hard to resolve..example - rainy weather, mediocre gps chip on phone due to which it takes a long time for satellite signal to get latched. 

To overcome these issues, we need to calculate approximate location using cell id. 

Patches are welcome. Stay sharp. Stay safe.
